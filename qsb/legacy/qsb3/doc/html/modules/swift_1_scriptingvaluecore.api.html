Module swift_1_scriptingvaluecore.api

Das Modul stellt grundlegende Funktionen zur Manipulation von Scripting Values bereit.

Vorausgesetzte Module:

  • (0) Core

API.GetEntityScale (_Entity)
Gibt den Größenfaktor des Entity zurück.

Der Faktor gibt an, um wie viel die Größe des Entity verändert wurde, im Vergleich zur normalen Größe. Faktor 1 entspricht der normalen Größe.
