Module swift_2_knighttitlerequirements.requirements

Definiert andere Aufstiegsbedingungen für den Spieler.

Originalfunktionen

Originalfunktionen

InitKnightTitleTables ()
Diese Funktion muss entweder in der QSB modifiziert oder sowohl im globalen als auch im lokalen Skript überschrieben werden. Bei Modifikationen muss das Schema für Aufstiegsbedingungen und Rechtevergabe immer beibehalten werden.

Hinweis: Diese Funktion wird automatisch vom Code ausgeführt. Du rufst sie niemals selbst auf!
