Module swift_1_inputoutputcore.api

Modul für die Eingabe durch den Spieler und die Ausgabe von Texten.

Du kannst vordefinierte Farben in Textausgaben verwenden. Außerdem kannst du für Skriptnamen und Entitytypen Platzhalter zu definieren. Diese Platzhalter können auch lokalisiert werden.

Du kannst ferner verschiedene Input Dialoge nutzen, um Ausgaben anzuzeigen. Außerdem kannst du vom Spieler Eingaben fordern.

Befehle:
Diese Befehle können über die Konsole (SHIFT + ^) eingegeben werden, wenn der Debug Mode aktiviert ist.

Befehl Beschreibung
clear Entfernt alle Texte aus dem Debug Window. (Nachrichten oben links)
version Zeigt die Versionsnummer der QSB an.
> Lua-Befehl in der globalen Umgebung ausführen.
>> Lua-Befehl in der lokalen Umgebung ausführen.
< Lua-Datei in die globale Umgebung laden.
<< Lua-Datei in die lokale Umgebung laden.

Vorausgesetzte Module:

Functions

API.DisableCheats ()
Deaktiviert die Cheats.

Hinweis: Die Cheats werden nur für den Spieler deaktiviert. Wenn der Debug Mode die Cheats aktiviert hat, bleiben sie aktiv.

Beispiel:

    API.DisableCheats();

Event

QSB.ScriptEvents
Events, auf die reagiert werden kann.
  • ChatOpened Das Chatfenster wird angezeigt (Parameter: PlayerID)
  • ChatClosed Die Chateingabe wird bestätigt (Parameter: Text, PlayerID)

Anwenderfunktionen

API.AddEntityTypePlaceholder (_Type, _Replacement)
Fügt einen Platzhalter für einen Entity-Typ hinzu.

Innerhalb des Textes wird der Plathalter wie folgt geschrieben: 

{t:ENTITY_TYP}
ENTITY_TYP muss mit einem Entity-Typ ersetzt werden. Der Typ wird ohne Entities. davor geschrieben.

Parameters:

  • _Type string Typname, der ersetzt werden soll
  • _Replacement string Wert, der ersetzt wird

Beispiel:

    API.AddNamePlaceholder("U_KnightHealing", "Arroganze Ziege");
API.AddNamePlaceholder("B_Castle_SE", {de = "Festung des Bösen", en = "Fortress of evil"});
API.AddNamePlaceholder (_Name, _Replacement)
Fügt einen Platzhalter für den angegebenen Namen hinzu.

Innerhalb des Textes wird der Plathalter wie folgt geschrieben: 

{n:SOME_NAME}
SOME_NAME muss mit dem Namen ersetzt werden.

Parameters:

  • _Name string Name, der ersetzt werden soll
  • _Replacement string Wert, der ersetzt wird

Beispiel:

    API.AddNamePlaceholder("Scriptname", "Horst");
API.AddNamePlaceholder("Scriptname", {de = "Kuchen", en = "Cake"});

API.ConvertPlaceholders (_Message)
Ersetzt alle Platzhalter im Text oder in der Table.

Mögliche Platzhalter:

  • {n:xyz} - Ersetzt einen Skriptnamen mit dem zuvor gesetzten Wert.
  • {t:xyz} - Ersetzt einen Typen mit dem zuvor gesetzten Wert.
  • {v:xyz} - Ersetzt mit dem Inhalt der angegebenen Variable. Der Wert muss in der Umgebung vorhanden sein, in der er verwendet wird. Das ist meistens das lokale Skript!

Außerdem werden einige Standardfarben ersetzt. 

{COLOR}
Ersetze {COLOR} in deinen Texten mit einer der gelisteten Farben.

API.DialogInfoBox (_PlayerID, _Title, _Text, _Action)
Öffnet einen Info-Dialog. Sollte bereits ein Dialog zu sehen sein, wird der Dialog der Dialogwarteschlange hinzugefügt.

An die Action wird der Spieler übergeben, der den Dialog bestätigt hat.

Hinweis: Kann nicht aus dem globalen Skript heraus benutzt werden.

Parameters:

  • _PlayerID string (Optional) Empfangender Spieler
  • _Title string Titel des Dialog
  • _Text string Text des Dialog
  • _Action Funktionsreferenz

Beispiel:

    API.DialogInfoBox("Wichtige Information", "Diese Information ist Spielentscheidend!");
API.DialogLanguageSelection (_PlayerID)
Öffnet den Dialog für die Auswahl der Sprache. Deutsch, Englisch und Französisch sind vorkonfiguriert.

Parameters:

  • _PlayerID number (optional) Nur für diesen Spieler anzeigen

Beispiel:

    -- Für alle Spieler
API.DialogLanguageSelection();
-- Nur für Spieler 2 anzeigen
API.DialogLanguageSelection(2);
API.DialogRequestBox (_PlayerID, _Title, _Text, _Action, _OkCancel)
Öffnet einen Ja-Nein-Dialog. Sollte bereits ein Dialog zu sehen sein, wird der Dialog der Dialogwarteschlange hinzugefügt.

Um die Entscheigung des Spielers abzufragen, wird ein Callback benötigt. Das Callback bekommt eine Boolean übergeben, sobald der Spieler die Entscheidung getroffen hat, plus die ID des Spielers.

Hinweis: Kann nicht aus dem globalen Skript heraus benutzt werden.

Parameters:

  • _PlayerID string (Optional) Empfangender Spieler
  • _Title string Titel des Dialog
  • _Text string Text des Dialog
  • _Action Funktionsreferenz
  • _OkCancel boolean Okay/Abbrechen statt Ja/Nein

Beispiel:

    function YesNoAction(_Yes, _PlayerID)
    if _Yes then GUI.AddNote("Ja wurde gedrückt"); end
end
API.DialogRequestBox("Frage", "Möchtest du das wirklich tun?", YesNoAction, false);
API.DialogSelectBox (_PlayerID, _Title, _Text, _Action, _List)
Öffnet einen Auswahldialog. Sollte bereits ein Dialog zu sehen sein, wird der Dialog der Dialogwarteschlange hinzugefügt.

In diesem Dialog wählt der Spieler eine Option aus einer Liste von Optionen aus. Anschließend erhält das Callback den Index der selektierten Option und die ID des Spielers, der den Dialog bestätigt hat.

Hinweis: Kann nicht aus dem globalen Skript heraus benutzt werden.

Parameters:

  • _PlayerID string (Optional) Empfangender Spieler
  • _Title string Titel des Dialog
  • _Text string Text des Dialog
  • _Action Funktionsreferenz
  • _List table Liste der Optionen

Beispiel:

    function OptionsAction(_Idx, _PlayerID)
    GUI.AddNote(_Idx.. " wurde ausgewählt!");
end
local List = {"Option A", "Option B", "Option C"};
API.DialogSelectBox("Auswahl", "Wähle etwas aus!", OptionsAction, List);

API.ShowTextInput (_PlayerID, _AllowDebug)
Bereitet die Texteingabe über den Chat Input vor.

Hinweis: Der Spieler kann den Input nicht mit Esc verlassen. Der Input kann nur durch Enter geschlossen werden. Das bedeutet, dass evtl. ein leerer String übergeben wird. In diesem Fall wurde nichts eingegeben.

Parameters:

  • _PlayerID number Für diesen Spieler anzeigen
  • _AllowDebug boolean Debug Commands auswerten

Beispiel:

    -- Debug Options werden geblockt
API.ShowTextInput(1, false);
-- Debug Options werden ausgewertet
API.ShowTextInput(1, true);
API.SimpleTextWindow (_Caption, _Content, _PlayerID)
Öffnet ein einfaches Textfenster mit dem angegebenen Text.

Die Länge des Textes ist nicht beschränkt. Überschreitet der Text die Größe des Fensters, wird automatisch eine Bildlaufleiste eingeblendet.
