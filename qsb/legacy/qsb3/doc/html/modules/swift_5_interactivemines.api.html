Module swift_5_interactivemines.api

Der Mapper kann eine Stein- oder Eisenmine restaurieren, die zuerst durch Begleichen der Kosten aufgebaut werden muss, bevor sie genutzt werden kann.


Optional kann die Mine einstürzen, wenn sie ausgebeutet wurde.

Vorausgesetzte Module:

  • (1) Jobs Core
  • (2) Interaction

Event

QSB.ScriptEvents Events, auf die reagiert werden kann.

Anwenderfunktionen

API.CreateIOIronMine (_Data) Erstelle eine verschüttete Eisenmine.
API.CreateIOStoneMine (_Data) Erstelle eine verschüttete Steinmine.


Event

QSB.ScriptEvents
Events, auf die reagiert werden kann.
  • InteractiveMineActivated Der Spieler aktiviert eine interaktive Mine (Parameter: ScriptName, KnightID, PlayerID)
  • InteractiveMineDepleted Eine ehemals interaktive Mine wurde ausgebeutet (Parameter: ScriptName)

Anwenderfunktionen

API.CreateIOIronMine (_Data)
Erstelle eine verschüttete Eisenmine.

Werden keine Materialkosten bestimmt, benötigt der Bau der Mine 500 Gold und 20 Holz.

Die Parameter der interaktiven Mine werden durch ihre Beschreibung festgelegt. Die Beschreibung ist eine Table, die bestimmte Werte für das Objekt beinhaltet. Dabei müssen nicht immer alle Werte angegeben werden.

Mögliche Angaben:

Feldname Typ Beschreibung Optional
Position string Der Skriptname des Entity, das zum interaktiven Objekt wird. nein
Title string Angezeigter Titel der Beschreibung für die Mine ja
Text string Angezeigte Text der Beschreibung für die Mine ja
Costs table Eine Table mit dem Typ und der Menge der Kosten. (Format: {Typ, Menge, Typ, Menge}) ja
ResourceAmount number Menge an Rohstoffen nach der Aktivierung ja
RefillAmount number Menge an Rohstoffen, die ein Geologe auffüllt (0 == nicht nachfüllbar) ja
ConstructionCondition function Eine zusätzliche Aktivierungsbedinung als Funktion. ja
ConstructionAction function Eine zusätzliche Aktion nach der Aktivierung. ja

Parameters:

  • _Data table Datentabelle der Mine

Verwandte Themen:

    API.CreateIOStoneMine

Beispiel:

  • -- Beispiel #1: Eine einfache Mine
API.CreateIOIronMine{
    Position = "mine"
};
  • -- Beispiel #2: Mine mit geänderten Kosten
API.CreateIOIronMine{
    Position = "mine",
    Costs    = {Goods.G_Wood, 15}
};
  • -- Beispiel #3: Mine mit Aktivierungsbedingung
API.CreateIOIronMine{
    Position              = "mine",
    Costs                 = {Goods.G_Wood, 15},
    ConstructionCondition = function(_Data)
        return HeroHasShovel == true;
    end
};
