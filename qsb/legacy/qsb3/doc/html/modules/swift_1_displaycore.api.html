Module swift_1_displaycore.api

Dieses Modul bietet rudimentäre Funktionen zur Veränderung des Interface und einen allgemeinen Black Screen für die Darstellung verschiedener Effekte.

Vorausgesetzte Module:

  • (0) Core

Event

QSB.ScriptEvents Events, auf die reagiert werden kann.

Anwenderfunktionen

API.ActivateBorderScroll (_PlayerID) Akliviert border Scroll wieder und löst die Fixierung auf ein Entity auf.
API.ActivateColoredScreen (_PlayerID, _Red, _Green, _Blue, _Alpha) Blendet einen farbigen Hintergrund über der Spielwelt aber hinter dem Interface ein.
API.ActivateImageScreen (_PlayerID, _Image, _Red, _Green, _Blue, _Alpha) Blendet eine Graphic über der Spielwelt aber hinter dem Interface ein.
API.ActivateNormalInterface (_PlayerID) Zeigt das normale Interface an.
API.DeactivateBorderScroll (_PlayerID, _Position) Deaktiviert Randscrollen und setzt die Kamera optional auf das Ziel
API.DeactivateColoredScreen (_PlayerID) Deaktiviert den farbigen Hintergrund, wenn er angezeigt wird.
API.DeactivateImageScreen (_PlayerID) Deaktiviert ein angezeigtes Bild, wenn dieses angezeigt wird.
API.DeactivateNormalInterface (_PlayerID) Blendet das normale Interface aus.
API.FinishCinematicEvent (_Name) Propagiert das Ende des Kinoeventss.
API.GetCinematicEventStatus (_Identifier) Gibt den Status des Kinoevents zurück.
API.IsCinematicEventActive (_PlayerID) Prüft ob gerade ein Kinoevents für den Spieler aktiv ist.
API.StartCinematicEvent (_Name, _PlayerID) Propagiert den Beginn des Kinoevents und bindet es an den Spieler.


Event

QSB.ScriptEvents
Events, auf die reagiert werden kann.
  • CinematicActivated Der Kinomodus wurde aktiviert (Parameter: KinoEventID, PlayerID)
  • CinematicConcluded Der Kinomodus wurde deaktiviert (Parameter: KinoEventID, PlayerID)
  • BorderScrollLocked Scrollen am Bildschirmrand wurde gesperrt (Parameter: PlayerID)
  • BorderScrollReset Scrollen am Bildschirmrand wurde freigegeben (Parameter: PlayerID)
  • GameInterfaceShown Die Spieloberfläche wird angezeigt (Parameter: PlayerID)
  • GameInterfaceHidden Die Spieloberfläche wird ausgeblendet (Parameter: PlayerID)
  • BlackScreenShown Der schwarze Hintergrund wird angezeigt (Parameter: PlayerID)
  • BlackScreenHidden Der schwarze Hintergrund wird ausgeblendet (Parameter: PlayerID)

API.ActivateImageScreen (_PlayerID, _Image, _Red, _Green, _Blue, _Alpha)
Blendet eine Graphic über der Spielwelt aber hinter dem Interface ein. Die Grafik muss im 16:9-Format sein. Bei 4:3-Auflösungen wird links und rechts abgeschnitten.

Parameters:

  • _Image string Pfad zur Grafik
  • _Red number (Optional) Rotwert (Standard: 255)
  • _Green number (Optional) Grünwert (Standard: 255)
