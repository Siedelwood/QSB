Module swift_5_interactivechests.api

Es werden Schatztruhen mit zufälligem Inhalt erzeugt.

Der Schatz einer Kiste oder Ruine wird nach Aktivierung in einem Karren abtransportiert.

Die erzeugten Truhen und Ruinen verhalten sich wie Interaktive Objekte. Werden ihnen Aktionen und Bedingungen mitgegeben, gelten für diese Funktionen die gleichen Regeln wie bei Interaktiven Objekten.

Vorausgesetzte Module:

  • (1) Interaction

Event

QSB.ScriptEvents Events, auf die reagiert werden kann.

Anwenderfunktionen

API.CreateRandomChest (_Name, _Good, _Min, _Max, _Condition, _Action)
Erstellt eine Schatztruhe mit einer zufälligen Menge an Waren des angegebenen Typs.

Die Menge der Ware ist dabei zufällig und liegt zwischen dem Minimalwert und dem Maximalwert.

Parameters:

  • _Name string Name der zu ersetzenden Script Entity
  • _Good number Warentyp
  • _Min number Mindestmenge
  • _Max number (Optional) Maximalmenge
  • _Condition number (Optional) Bedingung zur Aktivierung
  • _Action number (Optional) Aktion nach Aktivierung

Beispiel:

  • -- Bepspiel #1: Normale Truhe
API.CreateRandomChest("well1", Goods.G_Gems, 100, 300);
  • -- Bepspiel #2: Truhe mit Aktion
-- Wird die Bedingung weggelassen, tritt die Aktion an ihre Stelle
API.CreateRandomChest("well1", Goods.G_Gems, 100, 300, MyActionFunction);
  • -- Bepspiel #3: Truhe mit Bedingung
-- Wenn eine Bedingung gebraucht wird, muss eine Aktion angegeben werden.
API.CreateRandomChest("well1", Goods.G_Gems, 100, 300, MyConditionFunction, MyActionFunction);
