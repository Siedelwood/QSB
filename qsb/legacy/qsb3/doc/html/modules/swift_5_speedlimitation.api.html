Module swift_5_speedlimitation.api

Dieses Modul erlaubt die maximale Beschleunigung des Spiels zu steuern.

Anwenderfunktionen

API.SpeedLimitActivate (_Flag) Setzt die Spielgeschwindigkeit auf Stufe 1 fest oder gibt sie wieder frei.


Anwenderfunktionen

API.SpeedLimitActivate (_Flag)
Setzt die Spielgeschwindigkeit auf Stufe 1 fest oder gibt sie wieder frei.

Hinweis: Die Geschwindigkeitsbeschränkung wirkt sich ebenfalls auf Cheats aus. Es ist generell nicht mehr möglich, das Spiel zu beschleunigen, wenn die "Speedbremse" aktiv ist.

Parameters:

  • _Flag boolean Speedbremse ist aktiv

Beispiel:

    -- Geschwindigkeit auf Stufe 1 festsetzen
API.SpeedLimitActivate(true);
-- Geschwindigkeit freigeben
API.SpeedLimitActivate(false);
