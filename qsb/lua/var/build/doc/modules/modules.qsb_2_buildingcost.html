Suche

BCS.EditBuildingCosts (_UpgradeCategory, _Amount1, _Good2, _Amount2, _AddToBaseCost)
Überschreibt die Baukosten eines Gebäudes

Parameters:

  • _UpgradeCategory number UpgradeKategorie des Gebäudes, zB UpgradeCategories.BroomMaker
  • _Amount1 number Die neue Menge des ersten Rohstoffs (bei nil wird gelöscht)
  • _Good2 number (Optional) Der neue zweite Rohstoff der für den Bau bezahlt werden soll
  • _Amount2 number (Optional) Die Menge des zweiten Rohstoffs
  • _AddToBaseCost boolean (Optional) Sollen die angegebenen Kosten für den Originalrohstoff auf die Originalkosten aufgerechnet werden?

Verwandte Themen:

    BCS.SetUpgradeCosts

Beispiel:

    -- Neue Ausbaukosten definieren
BCS.SetUpgradeCosts(UpgradeCategories.BroomMaker, 100, Goods.G_Gold, 100)
-- Auf Originalkosten zurücksetzen
BCS.SetUpgradeCosts(UpgradeCategories.BroomMaker, nil)

BCS.SetConstructionAddedGoodDiscountFunction (_Function, _CanBeZero)
Übergibt eine Rabattfunktion für die Kosten des zweiten Rohstoffs für den Bauen von Gebäuden. Diese gelten nur für angepasste Baukosten und können nicht mehr zurückgenommen werden.

Parameters:

  • _Function number Rabattfunktion die einen Wert zwischen 0 und 1 zurückgibt
  • _CanBeZero boolean Soll der zweite Rohstoff auf Null gerundet werden können?

Verwandte Themen:

    BCS.SetConstructionOriginalGoodDiscountFunction

Beispiel:

    -- Der Spieler bekommt 20 Prozent Rabatt auf die Kosten des zweiten Rohstoffs beim Bau von Gebäuden.
-- Die Kosten des zweiten Rohstoffs wird werden immer mindestens 1 sein.
BCS.SetConstructionAddedGoodDiscountFunction(function()
    return 0.2
end, false)
