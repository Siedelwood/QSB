QSB.TraderTypes
Typen der Händler
  • GoodTrader Es werden Güter verkauft
  • MercenaryTrader Es werden Söldner verkauft
  • EntertainerTrader Es werden Entertainer verkauft
  • Unknown Unbekannter Typ (Fehler)

Anwenderfunktionen

API.AddEntertainerOffer (_VendorID, _OfferType)
Lässt einen NPC-Spieler einen Entertainer anbieten.

Parameters:

  • _VendorID number Spieler-ID des Verkäufers
  • _OfferType number Typ des Entertainer

Beispiel:

    -- Spieler 2 bietet einen Feuerschlucker an
API.AddEntertainerOffer(2, Entities.U_Entertainer_NA_FireEater);
API.AddGoodOffer (_VendorID, _OfferType, _OfferAmount, _RefreshRate)
Lässt einen NPC-Spieler Waren anbieten.

Parameters:

  • _VendorID number Spieler-ID des Verkäufers
  • _OfferType number Typ der Angebote
  • _OfferAmount number Menge an Angeboten
  • _RefreshRate number (Optional) Regenerationsrate des Angebot

Beispiel:

    -- Spieler 2 bietet Brot an
API.AddGoodOffer(2, Goods.G_Bread, 1, 2);
API.AddMercenaryOffer (_VendorID, _OfferType, _OfferAmount, _RefreshRate)
Lässt einen NPC-Spieler Söldner anbieten.

Hinweis: Stadtlagerhäuser können keine Söldner anbieten!

Parameters:

  • _VendorID number Spieler-ID des Verkäufers
  • _OfferType number Typ der Söldner
  • _OfferAmount number Menge an Söldnern
  • _RefreshRate number (Optional) Regenerationsrate des Angebot

Beispiel:

    -- Spieler 2 bietet Sölder an
API.AddMercenaryOffer(2, Entities.U_MilitaryBandit_Melee_SE, 1, 3);
API.GetOfferCount (_PlayerID)
Gibt die Menge an Angeboten im Lagerhaus des Spielers zurück. Wenn der Spieler kein Lagerhaus hat, wird 0 zurückgegeben.

Parameters:

  • _PlayerID number Player ID

Rückgabe:

    number Anzahl angebote

Beispiel:

    -- Angebote von Spieler 5 zählen
local Count = API.GetOfferCount(5);
API.GetOfferInformation (_PlayerID)
Gibt die Angebotsinformationen des Spielers aus. In dem Table stehen ID des Spielers, ID des Lagerhaus, Menge an Angeboten insgesamt und alle Angebote der Händlertypen.

Parameters:

  • _PlayerID number Player ID

Rückgabe:

    table Angebotsinformationen

Beispiel:

    local Info = API.GetOfferInformation(2);

-- Info enthält:
-- Info = {
--      Player = 2,
--      Storehouse = 26796.

--          {0, 0, Goods.G_Gems, 9, 2},
--          {0, 1, Goods.G_Milk, 9, 4},
--      },
-- };

  • _GoodOrEntityType number Warentyp oder Entitytyp

API.IsGoodOrUnitOffered (_PlayerID, _GoodOrEntityType)
Gibt zurück, ob das Angebot vom angegebenen Spieler im Lagerhaus zum Verkauf angeboten wird.

Parameters:

  • _PlayerID number Player ID
  • _GoodOrEntityType number Warentyp oder Entitytyp

Rückgabe:

    boolean Ware wird angeboten

Beispiel:

    -- Wird die Ware angeboten?
if API.IsGoodOrUnitOffered(4, Goods.G_Bread) then
    Logic.DEBUG_AddNote("Brot wird von Spieler 4 angeboten.");
end
API.ModifyTradeOffer (_PlayerID, _GoodOrEntityType, _NewAmount)
Ändert die aktuelle Menge des Angebots im Händelrgebäude.

Es kann ein beliebiger positiver Wert gesetzt werden. Es gibt keine Beschränkungen.

Hinweis: Wird eine höherer Wert gesetzt, als das ursprüngliche Maximum, regenerieren sich die Angebote nicht, bis die zusätzlichen Angebote verkauft wurden.

Parameters:

  • _PlayerID number Player ID
  • _GoodOrEntityType number ID des Händlers im Gebäude
  • _NewAmount number Neue Menge an Angeboten

Beispiel:

  • -- Beispiel #1: Angebote voll auffüllen
API.ModifyTradeOffer(7, Goods.G_Cheese, -1);
  • -- Beispiel #2: Angebote auffüllen
API.ModifyTradeOffer(7, Goods.G_Dye, 2);
