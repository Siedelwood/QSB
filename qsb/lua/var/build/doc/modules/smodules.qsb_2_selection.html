Module (2) Selektion

Die Optionen für selektierte Einheiten können individualisiert werden.

Es wird ein Button zum Entlassen von Einheiten hinzugefügt.

EinheitentypVorsteinstellung
Soldatenaktiv
Kriegsmaschinenaktiv
Diebedeaktiviert

Trebuchets haben nun das gleiche Menü, wie die anderen Kriegsmaschinen. Sie können jedoch nicht abgebaut werden.

Vorausgesetzte Module:

  • (0) Basismodul
  • (1) Anzeigekontrolle

Event

QSB.ScriptEvents Events, auf die reagiert werden kann.

Anwenderfunktionen

API.DisableReleaseSiegeEngines (_Flag) Deaktiviert oder aktiviert das Entlassen von Kriegsmaschinen.
API.DisableReleaseSoldiers (_Flag) Deaktiviert oder aktiviert das Entlassen von Soldaten.
API.DisableReleaseThieves (_Flag) Deaktiviert oder aktiviert das Entlassen von Dieben.
API.GetSelectedEntities (_PlayerID) Gibt alle selektierten Entities zurück.
API.GetSelectedEntity (_PlayerID) Gibt die ID des selektierten Entity zurück.
API.IsEntityInSelection (_Entity, _PlayerID) Prüft ob das Entity selektiert ist.


Event

QSB.ScriptEvents
Events, auf die reagiert werden kann.
  • SelectionChanged Die Selektion hat sich geändert (Parameter: PlayerID, OldIdList, NewIdList)

Anwenderfunktionen

API.DisableReleaseSiegeEngines (_Flag)
Deaktiviert oder aktiviert das Entlassen von Kriegsmaschinen.

Parameters:

  • _Flag boolean Deaktiviert (false) / Aktiviert (true)

Beispiel:

    API.DisableReleaseSiegeEngines(true);
API.DisableReleaseSoldiers (_Flag)
Deaktiviert oder aktiviert das Entlassen von Soldaten.

Parameters:

  • _Flag boolean Deaktiviert (false) / Aktiviert (true)

Beispiel:

    API.DisableReleaseSoldiers(false);
API.DisableReleaseThieves (_Flag)
Deaktiviert oder aktiviert das Entlassen von Dieben.

Parameters:

  • _Flag boolean Deaktiviert (false) / Aktiviert (true)

Beispiel:

    API.DisableReleaseThieves(false);
API.GetSelectedEntities (_PlayerID)
Gibt alle selektierten Entities zurück.

Parameters:

  • _PlayerID number ID des Spielers

Rückgabe:

    table ID des selektierten Entities

Beispiel:

    local Selection = API.GetSelectedEntities(1);
API.GetSelectedEntity (_PlayerID)
Gibt die ID des selektierten Entity zurück.

Wenn mehr als ein Entity selektiert sind, wird das erste Entity zurückgegeben. Sind keine Entities selektiert, wird 0 zurückgegeben.

Parameters:

  • _PlayerID number ID des Spielers

Rückgabe:

    number ID des selektierten Entities

Beispiel:

    local SelectedEntity = API.GetSelectedEntity(1);
API.IsEntityInSelection (_Entity, _PlayerID)
Prüft ob das Entity selektiert ist.

Parameters:

  • _Entity Entity das selektiert sein soll (Skriptname oder ID)
  • _PlayerID number ID des selektierenden Spielers

Rückgabe:

    boolean Entity ist selektiert

Beispiel:

    if API.IsEntityInSelection("hakim", 1) then
    -- Do something
end
